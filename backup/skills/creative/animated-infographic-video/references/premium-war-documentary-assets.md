- `middle east map` → base geography, Gulf orientation, political color maps, chokepoint map options
- `fighter jet vector` → strong cold-open or military-pressure scene hero asset
- `soldier silhouette vector` → immediate documentary tension in opening frames
- `missile icon` → weapons/methods scene, category badges, military toolkit panels
- `dubai skyline vector` → UAE consequence or closing future-of-UAE scene

## Composition lessons from this session
- For stronger first-3-second retention, open with **one headline + soldiers + aircraft + alert/shock shapes**, not a slow map build.
- Use maps as the explanation backbone after the hook, not as the only visual language.
- When the user says animation feels random, assign each scene a single motion purpose only:
  - hook fly-in
  - strike marker pop
  - route-line reveal
  - ship movement through Hormuz
  - skyline rise for UAE outlook
- Darker cinematic grading and stronger contrast work better for this war-documentary class than softer infographic palettes.

## Practical note
Direct unauthenticated HTTP requests to Magnific/Freepik-family pages may return 403, while the logged-in Magnific browser session can still browse and download the assets. Prefer the authenticated Magnific session first.
