# Hook-first character-led documentary rebuilds

Use this reference when the user rejects a war / geopolitics explainer for being unattractive, too map-only, or too ambient in its motion.

## Trigger phrases
- "This is not attractive at all"
- "Don't use just maps"
- "Use characters, army soldiers, fighter jets"
- "Hook the watcher in the first 3 seconds"
- "Animations are going here and there"
- "Make it more to the point"

## Durable visual response
When these signals appear, do not merely restyle the same map sequence. Rebuild around a **character-led documentary asset mix**:
- soldier silhouettes / troops
- fighter jets / aircraft
- missile or drone sheets for weapons-category framing
- one or two clean regional maps for geography only
- chokepoint map (for example Hormuz) for the key mechanism scene
- skyline or city silhouette for consequence / future scenes

Maps should remain the geographic backbone, but not the entire visual language.

## Recommended scene structure for this class
For a ~60–75 second documentary short:
1. **Cold open (0–3s)**
   - One dramatic headline only
   - Soldier / aircraft hero composition
   - 2–3 short badges max
   - No slow setup, no paragraph panel
2. **Strike geography (3–14s)**
   - Clean regional map
   - 3–5 markers max
   - One supporting panel only
3. **Weapons / pressure methods (14–28s)**
   - Fighter jet + missile / drone visual
   - Explain categories, not overclaimed exact models
4. **Chokepoint mechanism (28–42s)**
   - Strait / corridor map
   - Ship route reveal or single route animation
   - Explain disruption risk in plain language
5. **Regional consequence scene (42–58s)**
   - Skyline / city silhouette + short risk cards
   - Keep bullets concrete: shipping, flights, insurance, trade
6. **Future / takeaway (58s+)**
   - Pressure vs opportunity split
   - Finish on one clear strategic question

## Motion rules
When the user says animation feels random or "going here and there":
- only one dominant motion idea per scene
- use route reveals, marker pops, one fly-in, one pulse, or one skyline rise
- avoid stacking ambient drift, pulsing rings, ships, labels, and camera moves all at once
- remove decorative motion if it does not advance the narration
- treat stillness as a design tool; not every scene needs constant movement

## Asset sourcing pattern that worked
In this session, a logged-in Magnific session was able to provide usable premium assets even though direct public HTTP requests to Magnific/Freepik-family pages were blocked by 403.

Useful search classes inside the authenticated session:
- `fighter jet vector`
- `soldier silhouette vector`
- `missile icon vector`
- `middle east map`
- `dubai skyline vector`
- chokepoint-specific map searches where relevant

Practical workflow:
1. Search inside the authenticated Magnific browser session.
2. Open one promising asset.
3. Trigger download.
4. Recover the downloaded file from `/tmp/playwright-artifacts-*`.
5. Inspect whether it is JPG/ZIP/EPS.
6. Extract JPG previews from ZIPs for immediate motion-graphics use.

## Voiceover guidance for this class
- Keep the script shorter and harder-hitting than a standard infographic explainer.
- The first line should sound like a trailer hook, not an analyst memo.
- Good pattern: headline sentence -> why it matters -> map logic -> regional consequence -> final strategic question.

## TTS review workflow for documentary channels
When the user is selecting a voice for documentary / storytelling channels:
- generate several short sample clips instead of only naming voices
- prioritize deeper male voices first when requested
- send multiple options in a single batch for A/B comparison
- restore the user's previous default voice afterward so experimentation does not silently change future outputs

## Pitfalls
- Reusing the same map-heavy layout after the user explicitly asked for characters / soldiers / jets
- Letting the hook spend its first seconds on explanation instead of drama
- Using too many simultaneous micro-animations in one frame
- Making the geography scene as visually dense as the hook scene
- Using full military asset sheets without cropping / compositional control
